# راهنمای مطالعه LPIC-1
##بخش اول - بر مبنای سرفصل‌های آزمون 101
تهیه شده توسط شرکت «سیوک» با همکاری جمعی از دانش آموختگان دوره‌های LPI

با توجه قدیمی بودن کتاب‌های آموزش لینوکس موجود در بازار و نبود منبع مناسب برای مطالعه و یادگیری لینوکس، تصمیم گرفتیم شروع به تهیه یک مستند آنلاین برای این منظور کنیم. این مستند توسط داوطلبان تهیه شده است و به مرور زمان تکمیل تر می گردد. شما نیز می توانید به این پروژه کمک کنید. کافی است به صفحه [حمایت](/support.html) مراجعه کنید.
